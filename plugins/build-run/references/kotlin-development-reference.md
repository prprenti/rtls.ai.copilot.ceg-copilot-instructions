## Extension Recommendations

If the user encounters issues with Kotlin language features or mentions lack of IDE support, suggest installing the Kotlin extension:

**Installation Steps:**
1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac) to open the Command Palette
2. Type: `ext install mathiasfrohlich.Kotlin`
3. Press Enter to install

**Alternative Method:**
- Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac) to open Extensions view
- Search for "Kotlin"
- Install the "Kotlin Language" extension by mathiasfrohlich

## Signs Extension May Be Missing

You might need to recommend the extension if the user:
- Reports no syntax highlighting in `.kt` files
- Mentions missing code completion or IntelliSense
- Doesn't see error detection or diagnostics
- Cannot run or debug Kotlin code from VS Code

## Providing Assistance

When helping with Kotlin code:
- Provide idiomatic Kotlin solutions
- Use Kotlin-specific features (data classes, extension functions, coroutines, etc.)
- Follow Kotlin naming conventions (camelCase for functions/properties)
- Suggest null-safety best practices
- Recommend appropriate Kotlin standard library functions
